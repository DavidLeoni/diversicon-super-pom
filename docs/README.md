### Getting started

If you use Maven as build system, put this in the `dependencies` section of your `pom.xml`:

```xml
    <dependency>
        <groupId>eu.kidf</groupId>
        <artifactId>diversicon-super-pom</artifactId>
        <version>${project.version}</version>
    </dependency>
```
